Introduction & Explanation

This project compares alternative rock and pop music from the 1980s, 1990s, and 2000s, analyzing how the genre evolved in response to social, cultural, and technological changes. By exploring the lyrics the aim is to uncover similarities and differences between these genres over time. The project uses visual and textual analysis tools like Voyant and AntConc to track recurring themes, patterns, and words in song lyrics from each era.
